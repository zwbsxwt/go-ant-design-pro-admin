# Frontend Branding Contract

## User-Facing Product Name

The frontend MUST present the local product as:

```text
go-ant-design-pro-admin
```

## Required Surfaces

- Browser document title from `web/config/defaultSettings.ts`.
- Application title from `web/config/config.ts`.
- PWA manifest fields from `web/src/manifest.json`.
- Footer text in `web/src/components/Footer/index.tsx`.
- Login page brand text in `web/src/pages/user/login/index.tsx`.
- Starter copy in `web/src/pages/Welcome.tsx` and `web/src/pages/Admin.tsx`.

## Allowed Exceptions

- Package names and dependencies.
- Documentation that explicitly discusses Ant Design, Ant Design Pro,
  ProComponents, or Umi Max as upstream frameworks.
- URLs pointing to official Ant Design resources.
